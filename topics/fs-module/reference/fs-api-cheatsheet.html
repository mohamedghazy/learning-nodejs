Node.js fs Module — Quick Reference

The essential operations at a glance. All examples use node:fs/promises with ESM.

Imports

// Promise API (default for server code)
import { readFile, writeFile, appendFile, mkdir, rm, readdir, rename, stat, copyFile } from 'node:fs/promises';

// Streams (for large files)
import { createReadStream, createWriteStream } from 'node:fs';
import { pipeline } from 'node:stream/promises';

// File handles (byte-level access)
import { open } from 'node:fs/promises';

// Path utilities (always use with fs)
import { join, resolve, sep } from 'node:path';

Path Safety Pattern (Path Traversal Prevention)

import { resolve, sep } from 'node:path';

const BASE_DIR = resolve('uploads');

function safePath(userInput) {
  const target = resolve(BASE_DIR, userInput);
  if (!target.startsWith(BASE_DIR + sep) && target !== BASE_DIR) {
    throw new Error('Path traversal blocked');
  }
  return target;
}

Why + sep? Without it, /app/uploads-private passes startsWith('/app/uploads').

Security Checklist for File Endpoints

  1. resolve() + startsWith(base + sep) — path traversal
  2. realpath() + re-check — symlink escape
  3. Magic numbers — file type validation (don't trust extensions)
  4. Size limits — prevent resource exhaustion
  5. Sanitize filenames — strip special characters
  6. Don't leak paths in error responses

Stream Pipeline Pattern

import { createReadStream, createWriteStream } from 'node:fs';
import { pipeline } from 'node:stream/promises';
import { createGzip } from 'node:zlib';

// Read → Transform → Write (constant memory)
await pipeline(
  createReadStream('input.log'),
  createGzip(),
  createWriteStream('output.log.gz')
);
ScenarioUse
Small files (< 50MB)readFile / writeFile
Large/unknown size filesStreams + pipeline()
Line-by-line processingreadline.createInterface() + for await
Serving files over HTTPpipeline(readStream, transform, response)

Error Handling Pattern

try {
  const data = await readFile(path, 'utf8');
} catch (err) {
  if (err.code === 'ENOENT') { /* file missing */ }
  else if (err.code === 'EACCES') { /* permission denied */ }
  else throw err; // always re-throw unknowns
}
