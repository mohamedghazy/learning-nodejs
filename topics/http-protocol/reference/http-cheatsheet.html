HTTP Protocol & Node.js — Quick Reference

Status codes, headers, server patterns, CORS — all at a glance.

Basic Server

import http from 'node:http';
http.createServer((req, res) => {
  res.writeHead(200, { 'Content-Type': 'application/json' });
  res.end(JSON.stringify({ ok: true }));
}).listen(3000);

Request Object (req)

req.method'GET', 'POST', etc.
req.url'/users?page=2' (path + query)
req.headersObject (all keys lowercase)
req itselfReadable stream (body in chunks)

Response Object (res)

res.statusCode = 201Set status code
res.setHeader(name, value)Set individual header
res.writeHead(status, headers)Set status + headers at once
res.write(chunk)Write body chunk
res.end(data?)Send final chunk and close

URL Parsing

const url = new URL(req.url, `http://${req.headers.host}`);
url.pathname;                    // '/users'
url.searchParams.get('page');    // '2' (string!)

Read JSON Body

const chunks = [];
for await (const chunk of req) chunks.push(chunk);
const body = JSON.parse(Buffer.concat(chunks).toString());

CORS Headers

res.setHeader('Access-Control-Allow-Origin', 'https://myapp.com');
res.setHeader('Access-Control-Allow-Methods', 'GET,POST,PUT,DELETE');
res.setHeader('Access-Control-Allow-Headers', 'Content-Type,Authorization');
// Handle OPTIONS preflight: respond 204, no body
